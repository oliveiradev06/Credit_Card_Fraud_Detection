💳 Detecção de Fraudes em Cartões de Crédito

Esse projeto surgiu como forma de praticar análise de dados na prática, usando um dataset real de transações de cartão de crédito (disponível no Kaggle).

A ideia é entender melhor o comportamento das transações e tentar identificar padrões que possam indicar fraude.

O que já fiz até agora
Limpeza e tratamento dos dados com Pandas
Algumas análises exploratórias (EDA)
Criação de novas colunas pra ajudar na análise (como categorização de valores)
Identificação de desbalanceamento entre transações normais e fraudulentas
Criação de um dashboard simples usando Streamlit
O que ainda quero fazer
Aplicar modelos de machine learning pra detecção de fraude
Trabalhar melhor as métricas (principalmente por causa do desbalanceamento)
Melhorar o dashboard (deixar mais organizado e mais bonito)
Possivelmente subir o projeto online
